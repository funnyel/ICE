template Signal(Args...) ¶
Signal template mixin for the signals/slot system.
void emit(Args args) [public] ¶
Emit the signal (call all slots with given arguments).
void connect(void delegate(Args) deleg) [public] ¶
Connect a slot to the signal. The same slot can be connected more than once. This will result in multiple calls to that slot when the signal is emitted.
void disconnect(void delegate(Args) deleg) [public] ¶
Disconnect a slot from the signal. If a slot is connected more than once, it must be disconnected corresponding number of times.
