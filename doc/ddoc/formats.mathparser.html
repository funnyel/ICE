formats.mathparser

T parse_math(T)(string input, T[string] substitutions = null) ¶
Parse given string as a math expression.
An associative array of substitutions can be passed to substitute strings in the expression for numbers. Substitutions are not checked for operators or spaces, so it is possible to e.g. substitute "abc * d" for 42 . @return result of the expression.
