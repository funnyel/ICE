T reduce(alias fun, T)(T seed, T[] elems) ¶
Accumulating function.
For each element in array elems, expression seed = fun(seed, element) is evaluated. The resulting value of seed is returned. This is useful for e.g. computing the minimum value of an array, sum, etc.

Parameters:

fun(template alias) Function to process elems with.
seedStarting value of seed.
elemsArray to process.

Returns:

Resulting value of seed.
