ubyte[] background [public] ¶
Background color of the image. Can be empty.
Interpretation depends on color type.

Palette - background[0] is background color. Grayscale 8-bit - background[0] is background color. Grayscale 16-bit - background[0 .. 1] is background color. RGB 8-bit - background[0 .. 3] is background color. RGB 16-bit - background[0 .. 6] is background color. RGBA 8-bit - background[0 .. 4] is background color. RGBA 16-bit - background[0 .. 8] is background color.
