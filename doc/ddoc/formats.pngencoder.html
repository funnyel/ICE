Encode image data to PNG format.
Only 24bit RGB and 32bit RGBA formats are supported at the moment.

Parameters:

infoPNG information to encode.
sourceImage data to encode. Must correspond to width, height and color type in info.

Returns:

Manually allocated array with encoded PNG data. Must be manually freed.

Throws:

PNGException on failure.
