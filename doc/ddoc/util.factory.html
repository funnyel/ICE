util.factory

string generate_factory(string[] parameter_strings...) ¶
Generates data members and setters used in factory classes.
Each input string specifies a type, name and default value of a parameter, delimited by the '$' character. E.g.: 
1

"string $ width $ \"64\""
will result in generation of data member width_ (notice trailing underscore) of type string with default value of "64", and a setter for it, like this: 
1
2

protected string width_ = \"64\";
public void width(string width){width_ = width};

Parameters:

parameter_stringsArray of strings representing factory members, their types and default values.

Returns:

Generated code ready to be inserted into a factory class definition.
