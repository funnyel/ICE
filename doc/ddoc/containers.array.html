int find(T)(ref T[] array, bool delegate(ref T) deleg) ¶
Find an element in an array with a function.

Parameters:

arrayArray to search in.
delegFunction determining if this is the element we're looking for. Index of the first element for which this function returns true will be returned.

Returns:

Index of the element if found, -1 otherwise.

Examples:

1
2
3

int[] array = [1, 2, 1, 4, 3];
int i = array.find((ref int i){return i >= 3;}); //i is 3 (fourth element).
i = array.find((ref int i){return i > 4;});      //i is -1 (no element is greater than 4).

T min(T)(T[] array) ¶
Returns minimum value of an array.
Uses the > operator (opCmp for structs and classes) to get the minimum. If more than one element in the array is the minimum, any one of them could be returned.

Parameters:

arrayArray to find the minimum in. Must not be empty.

Returns:

Minimum of the array.

Examples:

1
2

int[] array = [1, 2, 1, 3];

Returns minimum value of an array using a function to compare elements.
If more than one element in the array is the minimum, any one of them could be returned.

Parameters:

arrayArray to find the minimum in. Must not be empty.
delegComparison function. Takes two parameters of the type stored in the array by reference and returns a bool that is true when the first parameter is greater than second, false otherwise.

Returns:
