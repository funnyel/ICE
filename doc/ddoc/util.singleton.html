void initialize(T)() [static] ¶
Initialize singleton with given subtype if it's not yet initialized.
This makes it possible to select implementation of a polymorphic singleton on runtime, e.g. if we have an abstract singleton class Interface and two implementations I1 and I2, we can initialize either using Interface.initialize!(I1) or Interface.initialize(!I2) to select implementation. This is useful for e.g. video driver.
