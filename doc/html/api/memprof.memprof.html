static YAMLNode allocationSizeDistribution(ref YAMLNode allocations, const size_t baseSize, const bool logarithmic);

Get a YAML mapping containing size distribution of allocations.

This categorizes allocations based on allocation size and gets the number of allocations in each category.

Parameters:
YAMLNode allocations YAML sequence of allocations to process.
size_t baseSize Used if logarithmic is false. Splits the distribution into equal sized categories sized baseSize bytes.
bool logarithmic If true, the distribution is split into categories based on powers of two.
static Node[] narrowAllocations(ref YAMLNode allocations, bool delegate(ref Node) predicate);

Narrow a sequence of YAML allocations to only those which satisfy predicate.

Parameters:
YAMLNode allocations YAML sequence of allocations to process.
bool delegate(ref Node) predicate Allocations for which this function returns true are kept; those for which it returns false are removed.
Returns:
Narrowed YAML sequence containing only allocations that satisfy predicate.
size_t max(T)(ref YAMLNode allocations, T delegate(ref YAMLNode) evaluate);

Get the maximum value returned by evaluate() for an allocation.

This computes evaluate(allocation) for each allocation and returns the maximum.

Parameters:
allocations YAML sequence of allocations to process.
evaluate Function to evaluate each allocation with.
Returns:
Maximum value of evaluate(allocation) from given sequence of allocations.
